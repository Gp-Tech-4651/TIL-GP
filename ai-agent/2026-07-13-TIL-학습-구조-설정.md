### Context
n8n 또는 AI 에이전트 개발 과정에서 효율적인 워크플로우를 구축하거나 프롬프트 엔지니어링을 최적화하기 위해 학습한 내용을 정리하고 공유하기 위한 구조입니다. 기술적인 문제 해결 과정이나 새로운 지식을 기록할 준비가 되었습니다.

### Core
학습 내용이나 문제 상황을 입력해주시면 다음과 같은 표준화된 포맷으로 정리해 드립니다.

*   **Context**: 문제 상황 또는 학습 배경
*   **Core**: 코드 스니펫, n8n JSON 설정, 또는 프롬프트 예시
*   **Insight**: 기술적 분석 및 참고 문서(출처 포함)

### Insight
기술 지식을 구조화하여 기록하는 것은 장기적인 문제 해결 능력을 향상시키는 데 큰 도움이 됩니다. 현재 궁금하신 기술적 주제나 정리하고 싶은 메모가 있다면 알려주세요.
